# 🧠 Typing Master in C

A simple yet fun **Typing Master** project written in pure **C language**, built to test and improve your typing speed and accuracy.  
Perfect for beginners learning C programming and exploring small console-based projects.

---

## 🚀 Features

- 🎯 Random sentence generation (25 motivational & coding-related lines)
- ⏱️ Calculates typing speed (WPM)
- ✅ Measures accuracy percentage
- 🧮 Displays total time taken
- 💻 Works on **Windows, macOS, and Linux**
- ⚙️ Beginner-friendly code (easy to understand & modify)
